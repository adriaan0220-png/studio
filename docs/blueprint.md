# **App Name**: Dorado Transport Solutions

## Core Features:

- Liquid Product Catalog: Showcase available liquid products for sale, with details such as type, origin, and specifications.
- Truck Fleet Overview: Display available trucks with their specifications, load capacity, and availability status.
- Delivery Location Input: Allow users to specify the desired delivery location using a map interface or address input.
- Generative Transport Cost Estimator: Estimate the transportation cost to the specified delivery location, considering factors such as distance, fuel costs, and truck availability. The LLM acts as a tool, providing prices depending on internal business metrics.
- Order Placement: Allow users to place orders for liquid products and/or truck transport services.
- Order Management: Track order status and manage deliveries.

## Style Guidelines:

- Primary color: Deep gold (#D4AF37), representing 'Dorado' and the valuable goods being transported.
- Background color: Light beige (#F5F5DC), a desaturated tint of the gold.
- Accent color: Brown (#A0522D), creating a visual harmony that suggests transport and earth.
- Body and headline font: 'PT Sans', a humanist sans-serif for modern readability.
- Use clear and simple icons to represent product types, truck features, and delivery options.
- Implement a clean and organized layout to showcase products, trucks, and delivery options clearly.
- Incorporate subtle animations for loading states and transitions to enhance user experience.
